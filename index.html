Modo prototipo

Carga datos inventados para probar el resumen mensual, los pagos fijos y el flujo de efectivo sin tocar Google Sheets.
Configuración
Conecta tu Google Sheet para que los datos de Telegram aparezcan automáticamente en el dashboard.
Instrucciones

Paso 1 — Google Sheet

Copia el ID de la URL de tu Sheet (la parte entre /d/ y /edit).


Paso 2 — Apps Script

En el Sheet: Extensiones → Apps Script → pega el código → Implementar como Web App.


Paso 3 — Make

Telegram Bot → Text Parser → Google Sheets. Ya está configurado y funcionando.


Paso 4 — Vercel

Sube esta carpeta a GitHub y conéctala a Vercel para tener la app en tu celular.
